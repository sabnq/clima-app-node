## Aplicación del clima en Node.js

Aplicación que mediante el uso de la consola de node.js e ingresando la ciudad o un lugar deseado, se obtiene la temperatura de dicho lugar.

Recuerden ejecutar el sigueinte comando antes de inicar la aplicación, para cargar las librerias

````
npm install
````

## Ejemplo:
Con la ciudad:

``
node app -d "Bogotá Colombia" 
``

Ó con una dirección especifica y la ciudad:

``
node app -d "Cr 88A 73 34, Bogotá"
``
